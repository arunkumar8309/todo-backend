## Clone the Repository

## git clone repository-url

https://github.com/arunkumar8309/todo-backend

cd todo-backend

## Install Dependencies

npm install

## .env File

MONGO_URI=mongodb://localhost:27017/todo
PORT=5000

## Start the Server Command

npm run start
